
# 🎮 Number Guessing Game

A simple and interactive Number Guessing Game built using HTML, CSS, and JavaScript.

## 🚀 Features

- Random number generation between 1 and 100
- Unlimited guessing attempts
- "Too High" and "Too Low" hints
- Attempt counter
- Guess history tracking
- Reset Game button
- Responsive and attractive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 How to Play

1. Enter a number between 1 and 100.
2. Click the **Guess** button.
3. The game will tell you:
   - 📈 Too High
   - 📉 Too Low
   - 🎉 Correct Guess
4. Keep guessing until you find the correct number.
5. Use **Reset Game** to start a new game.
